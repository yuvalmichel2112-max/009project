FlyTAU - airline management system 
created by group 009 
Features Based on the database schema
the system supports: User Management: Customers: registered, guests Employees: Managers. 
Flight Operations:
Scheduling Flights and assigning Airplanes and flight crew.
Bookings & Tickets:
Flight searching and booking.
Ticket issuance with seat selection (Economy/Business),
Booking status tracking (Active, Completed, Cancelled). 
Tech Stack:
Backend: Python (Flask) 
Database:  MySQL 
Frontend: HTML, CSS, JavaScript 
Hosting:  PythonAnywhere
